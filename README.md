# Awesome Zombie Smasher

3D Infinite Runner Game build with the Unity Game Engine and C# Programming Language.

![screenshot](./game_image.png)

## Built With
- Unity game engine
- C# for the scripts

### C# Libraries

- System.Collections
- System.Collections.Generic
- UnityEngine
- UnityEngine.UI
- UnityEngine.SceneManagement
